# Pack git

## Purpose
- Comandi Git rapidi e knowledge di base.

## Common Commands
- `dm -p git run gs` (git status)
- `dm -p git run gl` (log compatto)
- `dm -p git git-tools graph` (grafico completo)
- `dm -p git find branch` (cerca nei markdown)

## Notes
- Usa `dm pack use git` per renderlo default.
